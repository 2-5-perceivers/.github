# [2.5 Perceivers](https://2-5-perceivers.github.io/)

## Who are we?
Just 2 *and a half friends* who write code when we come up with ideas. It doesn't matter how good the ideas are, if it scratches an itch, it gets a repository.

We tend to build small tools, experiments, and the occasional thing that actually works.

Check out our **Warpinator projects** if you want to transfer files properly or you're just curious about **Rust** and file sharing protocols. Our **Flutter packages** might save you some time too. And if you're in a learning phase, **beTalk** is a good way to see the same thing done in way too many languages.

## Who are you?
No idea, but you might be the next person to open an issue, submit a PR, or tell us something is broken. All of those are welcome. We have Discussions enabled on some repositories if you'd rather just talk.
